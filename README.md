# AI Python Grundlagen

NLP Grundlagen und regelbasierte Chatbots aus dem KI-Python-Modul
meiner Weiterbildung zur **KI-Entwicklung** (August 2025).

## Inhalte

| Ordner | Beschreibung |
|---|---|
| `NLTKSetup.py` | Setup-Skript für NLTK mit deutschen Sprachressourcen |
| `spaCySetup.py` | Setup-Skript für spaCy mit deutschem Sprachmodell |
| `AIPythonUebung02601NPL-Grundlagen` | NLP Einstieg – Tokenisierung, Sentiment-Analyse, Lemmatisierung |
| `AIPythonUebung03201` | VADER Sentiment mit eigener Lexikon-Erweiterung und Async-Übersetzung |
| `AIPythonUebung03202` | Regelbasierter Restaurantbot mit Keyword-Matching |
| `AITransferaufgabe03401` | Vergleich regelbasierter vs. KI-basierter Chatbot-Ansatz |
| `CaseStudy04501` | Reservierungsbot mit spaCy NLP und Regex-Hybrid für Entity-Extraktion |
| `Transferaufgabe05501` | Online-Shop Kundenservice – TF-IDF, Logistische Regression und DistilBERT |

## Highlights
- Eigene VADER-Lexikon-Erweiterung für deutschen Slang
- Hybride Entity-Extraktion mit spaCy und Regex
- Synthetische Trainingsdaten mit Template-System
- DistilBERT-Klassifikation mit `simpletransformers`
- Hyperparameter-Vergleich und Datenqualitätsbericht

## Technologien
- **Sprache:** Python 3
- **Bibliotheken:** spaCy, NLTK, scikit-learn, simpletransformers, googletrans
- **Modelle:** de_core_news_sm, DistilBERT
- **IDE:** PyCharm
